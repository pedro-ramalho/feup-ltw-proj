# How to run

## Step 1

Clone the project:

```bash
$ git clone git@github.com:FEUP-LTW-2022/ltw-t01-g08.git
```

and enter that directory:

```bash
$ cd ltw-t01-g08
```

## Step 2

Launch the server on a localhost:
```bash
$ php -S localhost:9000
```

## Step 3

Open a web browser and open the server:
```
https://localhost:9000
```
